# Feeder Timer

Mobilbarat feeder timer alkalmazas Vite + React + TypeScript alapon.

## Fo funkciok

- allithato visszaszamlalo `30 mp` es `30 perc` kozott,
- inditas es nullazas,
- `0`-nal hangjelzes,
- `0` utan az ido tovabb szamol, amig nem nyomsz gombot,
- lejarat utan egy gombnyomassal visszaall a beallitott idore es azonnal ujraindul,
- nagy kontrasztu, egyszeru, kulteri mobilhasznalatra optimalizalt kezelofelulet.

## Fejlesztoi inditas

```bash
npm install
npm run dev
```

Ezutan nyisd meg a terminalban megjeleno helyi cimet.

## Build

```bash
npm run build
npm run preview
```
